Optimizations

Constant Folding

Evaluates constant expressions at compile time

let x = 5 + 3; ‚Üí let x = 8;

Dead Code Elimination

Removes unreachable code branches

if false { ... } ‚Üí removed

Constant Propagation

Replaces variables with known constant values

x = 5; y = x; ‚Üí y = 5;

Strength Reduction

Replaces expensive ops with cheaper ones

x * 2 ‚Üí x << 1

About MiniLang

MiniLang is a statically-typed compiled programming language built entirely in Rust. It features unique syntax with Pascal-style logical operators (AND/OR/NOT), Python-style comments (#), and distinctive keywords (display/send) that give it a unique identity.
